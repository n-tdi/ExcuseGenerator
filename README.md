# Excuse Generator
 Stuck in a awkward situation? Use the ExcuseGenerator to generate a random excuse to get out of it.
 
## Features
<ul>
    <li>Object Orientated</li>
    <li>Easy to use</li>
    <li>Open to the community to add their own response!</li>
</ul>
